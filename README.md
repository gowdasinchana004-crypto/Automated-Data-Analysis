# Autolysis Project

## Overview

This project performs automated exploratory data analysis using Python.

## Features

* Missing value detection
* Summary statistics
* Histogram visualizations
* Correlation matrix
* Outlier detection using boxplots

## Datasets Used

* happiness.csv
* media.csv
* goodreads.csv

## How to Run

```bash
uv run autolysis.py happiness.csv
```

## Output

The script generates:

* Histogram plots
* Correlation matrix
* Boxplots for outlier detection
